## About

ChessGame is a chess game built in Java. It implements the full rules of chess including move validation, check/checkmate detection, and a playable game loop. A compiled `.jar` is included so you can run it without building from source.

---

## Getting started

### Run the jar directly

```bash
java -jar ChessProject.jar
```

### Build from source

```bash
git clone https://github.com/Roycelando/ChessGame
cd ChessGame
javac src/**/*.java
```
